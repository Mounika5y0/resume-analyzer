# AI Resume Analyzer

An AI-powered Resume Analyzer built using HTML, CSS, and JavaScript.

## Features

- Upload PDF Resume
- Upload DOCX Resume
- Analyze Resume
- Display Resume Score
- Modern Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PDF.js
- Mammoth.js

## Project Structure

AI resume analyzer/
│── resume.html
│── resume.css
│── resume.js
│── README.md

## How to Run

1. Download or clone the project.
2. Open the folder in VS Code.
3. Open `resume.html` using Live Server.
